## Installation

1. create env
    e.g.: `python -m venv venv`
2. activate env
    `venv\Scripts\activate`
3. Install python + pytest
    `pip install pytest-playwright`
4. Install browsers
    `playwright install`
5. Install .dotenv
    `pip install python-dotenv`
6. run test
    e.g.: `pytest .\tests\test_orange.py --headed`
